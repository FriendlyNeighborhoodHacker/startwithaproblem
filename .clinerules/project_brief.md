### Project Brief
This folder is startwithaproblem.com ("Start With a Problem"), a web site focused on inspiring students to work on important problems in the world.  It is built on "Get Simple" CMS which is a file-system based CMS.

### Theme
The theme that it is using is "ResponsiveCE" (/theme/ResponsiveCE), and in that directory can be found most of the HTML / CSS for the site.
